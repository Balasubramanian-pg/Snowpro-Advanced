# Question 005

![Question 005](https://raw.githubusercontent.com/Balasubramanian-pg/Snowpro-Advanced/main/05%20Practice%20Questions/Test%20Exams/Assets/Screenshot%20%28176%29.png)

Pick these three:

* **A schema created with WITH MANAGED ACCESS restricts privilege grants** to the schema owner or roles with MANAGE GRANTS
* **Schemas can be moved between databases using `ALTER SCHEMA ... RENAME TO db.schema`**
* **A transient schema makes all tables within it transient by default**

Why the others are wrong:

* A schema **cannot exist independently** of a database
* The **PUBLIC schema is created automatically** with every new database, not manually
